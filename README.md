## Overview
This project is a Dockerized real-time WebSocket application built on a Django backend. It enables containerised execution environments for multiple programming languages including Python, Java, C++, C, and JavaScript. The system uses WebSockets for real-time communication and Docker containers for runtime isolation.


## Features
- Django-ASGI backend with Channels
- Real-time communication via WebSockets
- Containerised multi-language execution environments
- Docker-based runtime isolation
- Modular execution workflow
- IDE-integrated development

## Tech Stack
- Django
- WebSockets
- Docker
- Containerised runtimes:
  - Python
  - Java
  - C++
  - C
  - JavaScript
